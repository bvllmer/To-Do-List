## Features
- Save tasks to LocalStorage, restoring them on restart.
- Mark tasks as done.
- Remove tasks.
- View parameters.
  - ?view=all
  - ?view=active
  - ?view=completed
- Edit tasks inline.
- Drag & drop reordering.
- Clear all completed tasks.
